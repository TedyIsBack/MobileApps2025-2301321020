## **ExploreNow — Android приложение за управление на забележителности**
---
### 1. Идея

ExploreNow е Android приложение, което позволява на потребителите да създават, редактират, преглеждат и изтриват забележителности (Landmarks).
Всяка забележителност съдържа: **име,адрес, описание, снимка, QR код, генериран автоматично от данните**

*Приложението използва Room (SQLite ORM) за локално съхранение и предлага модерен Material 3 интерфейс с поддръжка на светла/тъмна тема.*

### 2.Как работи

* **MainActivity**: Показва всички запазени забележителности (RecyclerView)

   Всеки елемент има:
   * QR → отваря екрана за генериране на QR
   * Edit → отваря формата за редакция
   * Delete → изтрива записа

Ако няма записи → показва се празен екран с бутон **Add Landmark**

* **EditorActivity**
  * Формуляр с полета: Име, Адрес, Описание, Снимка (по избор)
  * Save → запис в локалната база данни
  * Поддържа Create + Update

* **LandmarkQrActivity** (QR генериране)
  * Генерира QR код за избраната забележителност с помощта на ZXing
  * QR се базира на сериализирана информация за Landmak под формата на линк с landmark id
* **Room** база данни
  * Entity: Landmark
  * DAO: CRUD операции
  * Repository: Абстракция за достъп до данните
  * AppDatabase: Singleton база



### 3. Архитектурен модел:
* MVVM + Repository
* Room (persistency)
* RecyclerView Adapter architecture
* QR utils (ZXing)

### 4.Потребителски поток
 Потребителят отваря приложението → вижда списък от забележителности
-> Натиска Add Landmark → отваря се EditorActivity
-> Въвежда информация → натиска Save
-> Записът се появява в основния списък 

*Данните остават в базата след рестарт*
